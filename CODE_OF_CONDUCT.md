# 行为准则

## 我们的承诺

我们致力于为每个人提供一个开放、友好、多元、包容和健康的协作环境。

## 我们的标准

积极行为包括：

- 使用友好、尊重和包容的语言
- 接纳不同背景、经验和观点
- 理性接受建设性反馈
- 优先考虑社区和项目的长期健康

不可接受的行为包括：

- 性暗示语言或图像
- 人身攻击、侮辱、贬低性评论
- 公开或私下骚扰
- 未经许可披露他人隐私或敏感信息

## 举报与执行

如果你遇到违反本准则的行为，请不要在公开线程中披露敏感细节。

可以通过以下方式联系维护者：

- 仓库所有者 GitHub 主页：`https://github.com/ShuaiShuai03`
- 如事件同时涉及安全漏洞，也可参考 [SECURITY.md](SECURITY.md)

所有投诉都会被审查，并根据情况采取适当措施。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 2.1 版。
